                               Курсовая работа.
                    Исследование функции одной переменной.

                             Работа с программой.
 1. Для построения графика необходимо нажать кнопку <Построить график>.
 2. Для  нахождения  нулей функции надо нажать кнопку <Поиск нулей>. Программа
    выдаст отрезки вместе со знаками, которые принимает на этих отрезках функ-
    ция.  Знак  0 означает, что функция возможно обращается в ноль в одной или
    нескольких точках отрезка.
 3. При вводе своих функций можно использовать операции +, -, *, /, а также ^
    (возведение в степень) и следующие элементарные функции:
       sqrt - квадратный корень,
       exp  - экспонента,
       ln   - натуральный логарифм.
    Независимая переменная - x.
 4. Задание для курсовой работы: f(x)=ln(1+x^2)+1/ln(2+x^2)-5.

Для запуска программы используйте файл kursovik.exe.
